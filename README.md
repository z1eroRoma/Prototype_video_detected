Table Detection Prototype

## Описание проекта

Данный проект представляет собой прототип системы компьютерного зрения для анализа состояния столиков по видео.

Цель системы — автоматически определять:

* находится ли столик в использовании
* когда столик освобождается

На основе этих данных рассчитывается метрика:

## Используемые технологии

* Python 3.8+
* OpenCV — работа с видео и визуализация
* Ultralytics YOLOv8 — детекция людей
* Pandas — обработка событий и аналитика
* NumPy — вычисления

## Принцип работы

### 1. Выбор зоны анализа (ROI)

При запуске пользователь вручную выделяет область столика на первом кадре видео.

---

### 2. Детекция людей

Для каждого кадра используется модель YOLOv8, которая определяет bounding box'ы людей.

---

### 3. Определение состояния столика

Система определяет, находится ли человек в зоне столика:

* **EMPTY** — столик пуст
* **OCCUPIED** — в зоне есть человек

---

### 4. Сглаживание (anti-flicker)

Для повышения стабильности реализованы:

* буфер кадров (temporal smoothing)
* минимальная длительность состояния

Это позволяет устранить ложные срабатывания и “дребезг” детекции.

---

### 5. Аналитика

На основе событий рассчитывается:

>  **Среднее время между уходом и следующим подходом**

## Ограничения

* Обрабатывается только один столик
* Нет трекинга конкретных людей
* Возможны ошибки при перекрытии объектов
* Работа зависит от качества видео
